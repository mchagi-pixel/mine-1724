# 💕 Manoj ❤️ Rachel - Love Card

A beautiful, interactive romantic love card created with HTML, CSS, and JavaScript.

## Features

✨ **Interactive Design**
- Animated beating heart on initial load
- Click to reveal romantic message and photo
- Floating heart animations throughout the page
- Confetti celebration effect
- Beautiful gradient background

💖 **Romantic Elements**
- Personalized love message
- Photo carousel gallery
- Heartfelt poetry and romantic words
- Smooth transitions and animations
- Music toggle for ambiance

🎨 **Modern UI**
- Glassmorphism design
- Responsive layout
- Smooth animations and transitions
- Beautiful color gradients

## How to Use

1. Open `2nd.html` in your web browser
2. Enjoy the initial animated heart
3. Click "Click Me 💕" to reveal your love card
4. Browse through the photos using the gallery controls
5. Toggle background music on/off
6. Celebrate with confetti effects

## Files

- `2nd.html` - Main love card application
- `image.jpg` - Your special photo
- `background-music.mp3` - Optional romantic background music
